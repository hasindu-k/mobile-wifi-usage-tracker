1. Persistence (The "Memory" Gap)
What's missing: A database (like Room).
User impact: Currently, if a user sets a "10 GB Limit" on the Limits screen, it will disappear the moment they close the app. There is no way to save preferences or track historical trends over months.
2. Active Monitoring & Notifications
What's missing: A Background Service or WorkManager.
User impact: A data monitor is only useful if it warns you before you overspend. The app needs to check usage in the background and send a push notification: "Warning: You've used 90% of your mobile data!"
